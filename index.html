CPU Scheduler
Step 01 — Algorithm
Select Scheduling Algorithm
Choose the CPU scheduling strategy. Non-preemptive algorithms run a process to completion; preemptive ones may interrupt it mid-execution.
FCFS
Non-Preemptive
First-Come, First-Served
Runs processes in arrival order. Simple but prone to the convoy effect.
SJF
Non-Preemptive
Shortest Job First
Picks the shortest burst-time process. Minimizes average waiting time.
SRT
Preemptive
Shortest Remaining Time
Preemptive SJF. Interrupts running process when a shorter job arrives.
RR
Preemptive
Round Robin
Each process gets a fixed time quantum cyclically. Fair time-sharing.
PRI-NP
Non-Preemptive
Priority (Non-Preemptive)
Picks highest-priority ready process and runs it to completion.
PRI-P
Preemptive
Priority (Preemptive)
Preempts running process when a higher-priority job arrives.
P-RR
Preemptive
Priority + Round Robin
Priority-based scheduling with Round Robin among equal-priority processes.
